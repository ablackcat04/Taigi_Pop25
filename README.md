## Taigi_Pop25 是什麼?

Taigi_Pop25 是一個專為歌聲合成系統製作的台語歌唱資料集，包含10首台語流行歌與1首補充資料，由一位女性歌手演唱。

錄音總時長約38分鐘，其中有效歌唱時常約為25分鐘。

所有錄音皆人工標註句子、字及音素(phoneme)。整個資料集一共有4413個phoneme標記，包含台語全17種聲母以及68種韻母，覆蓋絕大多數台語歌唱情境。

音高標記使用midi格式，所有midi檔案bpm皆為120。
